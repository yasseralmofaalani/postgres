المحاضرة 02: تثبيت وإعداد PostgreSQL

المستوى: المبتدئ المدة: 60 دقيقة
← العودة إلى الفهرس

متطلبات النظام

قبل تثبيت PostgreSQL، تأكد من أن نظامك يلبي المتطلبات التالية:

  • الذاكرة: 512 MB على الأقل (2 GB موصى به)
  • مساحة القرص: 1 GB على الأقل للتثبيت الأساسي
  • نظام التشغيل: Windows, Linux, أو macOS

التثبيت على Windows

الطريقة 1: استخدام المثبت الرسمي

  1. قم بتحميل PostgreSQL من الموقع الرسمي: postgresql.org
  2. شغّل ملف التثبيت (.exe)
  3. اتبع معالج التثبيت:
    • اختر مجلد التثبيت (الافتراضي: C:\Program Files\PostgreSQL)
    • اختر المكونات: PostgreSQL Server, pgAdmin, Command Line Tools
    • حدد مجلد البيانات
    • أدخل كلمة مرور المستخدم postgres
    • اختر المنفذ (الافتراضي: 5432)
  4. أكمل التثبيت وانتظر حتى ينتهي
التحقق من التثبيت
-- فتح PowerShell أو Command Prompt
psql --version

-- يجب أن ترى رقم الإصدار، مثلاً:
-- psql (PostgreSQL) 16.0

الطريقة 2: استخدام Chocolatey

تثبيت باستخدام Chocolatey
# تثبيت Chocolatey أولاً (إذا لم يكن مثبتاً)
# ثم قم بتشغيل:
choco install postgresql

# أو مع PostGIS:
choco install postgresql --params '/Password:YourPassword'

التثبيت على Linux

Ubuntu/Debian

تثبيت PostgreSQL على Ubuntu
# تحديث قائمة الحزم
sudo apt update

# تثبيت PostgreSQL
sudo apt install postgresql postgresql-contrib

# تثبيت PostGIS
sudo apt install postgis postgresql-16-postgis-3

# بدء الخدمة
sudo systemctl start postgresql
sudo systemctl enable postgresql

# التحقق من الحالة
sudo systemctl status postgresql

CentOS/RHEL

تثبيت PostgreSQL على CentOS
# تثبيت مستودع PostgreSQL الرسمي
sudo yum install -y https://download.postgresql.org/pub/repos/yum/reporpms/EL-7-x86_64/pgdg-redhat-repo-latest.noarch.rpm

# تثبيت PostgreSQL
sudo yum install -y postgresql16-server postgresql16

# تهيئة قاعدة البيانات
sudo /usr/pgsql-16/bin/postgresql-16-setup initdb

# بدء الخدمة
sudo systemctl start postgresql-16
sudo systemctl enable postgresql-16

التثبيت على macOS

استخدام Homebrew

تثبيت PostgreSQL على macOS
# تثبيت Homebrew (إذا لم يكن مثبتاً)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# تثبيت PostgreSQL
brew install postgresql@16

# تثبيت PostGIS
brew install postgis

# بدء الخدمة
brew services start postgresql@16

الإعداد الأولي

1. إنشاء قاعدة بيانات أولية

إنشاء قاعدة بيانات
# تسجيل الدخول كمستخدم postgres
sudo -u postgres psql

# أو على Windows (بعد إضافة PostgreSQL إلى PATH)
psql -U postgres

# إنشاء قاعدة بيانات جديدة
CREATE DATABASE mydb;

# إنشاء مستخدم جديد
CREATE USER myuser WITH PASSWORD 'mypassword';

# منح الصلاحيات
GRANT ALL PRIVILEGES ON DATABASE mydb TO myuser;

# الخروج
\q

2. تثبيت PostGIS

تفعيل PostGIS في قاعدة البيانات
# الاتصال بقاعدة البيانات
psql -U postgres -d mydb

# تثبيت PostGIS
CREATE EXTENSION postgis;

# التحقق من التثبيت
SELECT PostGIS_Version();

# يجب أن ترى رقم الإصدار، مثلاً:
-- POSTGIS="3.4.0" [EXTENSION] PGSQL="160" GEOS="3.12.0" PROJ="9.3.0"

3. إعداد ملف التكوين

ملف postgresql.conf يحتوي على إعدادات الخادم:

تعديل إعدادات PostgreSQL
# موقع الملف:
# Windows: C:\Program Files\PostgreSQL\16\data\postgresql.conf
# Linux: /etc/postgresql/16/main/postgresql.conf
# macOS: /usr/local/var/postgres/postgresql.conf

# إعدادات مهمة:
listen_addresses = '*'          # السماح بالاتصالات من أي عنوان
port = 5432                      # منفذ الاتصال
max_connections = 100            # الحد الأقصى للاتصالات
shared_buffers = 256MB           # الذاكرة المشتركة
effective_cache_size = 1GB       # حجم الذاكرة المؤقتة

4. إعداد ملف pg_hba.conf

هذا الملف يتحكم في المصادقة والاتصالات:

إعدادات المصادقة
# موقع الملف:
# Windows: C:\Program Files\PostgreSQL\16\data\pg_hba.conf
# Linux: /etc/postgresql/16/main/pg_hba.conf

# مثال على الإعدادات:
# TYPE  DATABASE  USER  ADDRESS  METHOD
host    all       all   0.0.0.0/0  md5
local   all       all              peer
تحذير: بعد تعديل ملفات التكوين، يجب إعادة تشغيل خدمة PostgreSQL لتطبيق التغييرات.

أدوات إدارة PostgreSQL

1. psql (سطر الأوامر)

أوامر psql الأساسية
# الاتصال بقاعدة البيانات
psql -U username -d database_name -h localhost

# الأوامر الداخلية لـ psql:
\l          # عرض قائمة قواعد البيانات
\c dbname   # الاتصال بقاعدة بيانات
\dt         # عرض الجداول
\d table    # وصف جدول
\q          # الخروج
\?          # مساعدة

2. pgAdmin (واجهة رسومية)

pgAdmin هو أداة إدارة رسومية قوية لـ PostgreSQL. يتم تثبيتها تلقائيًا مع PostgreSQL على Windows، ويمكن تثبيتها بشكل منفصل على أنظمة أخرى.

  • إدارة قواعد البيانات والجداول
  • تنفيذ الاستعلامات SQL
  • عرض البيانات والنتائج
  • إدارة المستخدمين والصلاحيات
  • النسخ الاحتياطي والاستعادة

التحقق من التثبيت

اختبار التثبيت
# 1. التحقق من إصدار PostgreSQL
psql --version

# 2. التحقق من حالة الخدمة
# Windows:
Get-Service postgresql*

# Linux:
sudo systemctl status postgresql

# macOS:
brew services list

# 3. الاتصال بقاعدة البيانات
psql -U postgres

# 4. التحقق من PostGIS
SELECT PostGIS_Version();
SELECT PostGIS_Full_Version();

حل المشاكل الشائعة

مشكلة: خطأ "connection refused"

الحل:

  • تأكد من أن خدمة PostgreSQL تعمل
  • تحقق من إعدادات postgresql.conf و pg_hba.conf
  • تأكد من أن المنفذ 5432 مفتوح
مشكلة: خطأ "password authentication failed"

الحل:

  • تأكد من كلمة المرور الصحيحة
  • تحقق من إعدادات المصادقة في pg_hba.conf

  • استخدام أدوات الإدارة (psql و pgAdmin)
